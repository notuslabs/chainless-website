# Logo System

> Phase: identity | Brand: chainless | Generated: 2026-03-22

---

## Existing Mark — Preserved

The Chainless logo is a preserved brand asset with proven recognition equity among existing users. This document defines how the existing mark operates within the evolved visual system — not a redesign, but a systematization that elevates the logo into its proper premium environment.

**Strategic rationale for preservation:** The Chainless name and mark carry the brand's most valuable semantic asset — "without chains" — which maps directly to the sovereignty-first positioning. The logo already communicates freedom and independence. What changes is the system around it: the typography, the spacing, the contexts, the companions.

---

## Logo Lockups

### Primary Lockup (Horizontal)

The full brand mark: icon + "Chainless" wordmark, arranged horizontally. Use this as the default wherever space allows.

- **Minimum width:** 120px (digital), 30mm (print)
- **Preferred placement:** Top-left of page/screen, centered in hero contexts
- **Color:** `#1C1B19` on light backgrounds, `#FAFAF8` on dark backgrounds

### Secondary Lockup (Stacked)

Icon above wordmark, centered alignment. Use when horizontal space is constrained or for square/portrait formats.

- **Minimum width:** 80px (digital), 20mm (print)
- **Use cases:** App icon frame, square social avatars, vertical signage

### Icon Only

The symbol mark without wordmark. Use only when brand context is already established (e.g., favicon, app icon, watermarks, repeat patterns).

- **Minimum size:** 24px (digital), 6mm (print)
- **Use cases:** Favicon (32×32), app icon (1024×1024 master), social profile pictures, loading states

### Monochrome Variations

| Variation | Usage | Color |
|-----------|-------|-------|
| **Dark mono** | Light backgrounds, print on white stock | `#1C1B19` |
| **Light mono** | Dark backgrounds, reversed contexts | `#FAFAF8` |
| **Single-color** | One-color print, embossing, engraving | Any single brand color |

### On Brand Yellow

The logo in `#1C1B19` on a `#FFC73D` background. This is the highest-impact brand expression — use sparingly for premium brand moments.

- **Use cases:** Business cards, brand merchandise, campaign hero moments, event signage
- **Rule:** Never place the yellow logo on a yellow background. Always dark mark on yellow ground.

## Clear Space

Clear space ensures the logo maintains visual authority. It is measured using the **cap-height** of the "C" in "Chainless" (referred to as **1×**).

| Context | Clear space (all sides) |
|---------|------------------------|
| **Primary lockup** | 2× minimum |
| **Secondary lockup (stacked)** | 1.5× minimum |
| **Icon only** | 1× minimum |
| **Hero placement (generous)** | 4× recommended |

No other logos, text, graphic elements, or container edges may enter the clear space zone.

## Responsive Sizing Guidelines

The logo scales with the medium. These are recommended sizing ranges, not strict rules — use judgment based on layout context.

| Context | Recommended logo width | Lockup |
|---------|:----------------------:|--------|
| **Website header (desktop)** | 140–160px | Primary horizontal |
| **Website header (mobile)** | 100–120px | Primary horizontal (reduced) or icon + short wordmark |
| **App splash screen** | 200–240px | Secondary stacked |
| **App header bar** | 28–32px icon | Icon only |
| **Social media profile** | 100% of container | Icon only (inside safe zone) |
| **Email signature** | 120–140px | Primary horizontal |
| **Print — A4 letterhead** | 35–45mm | Primary horizontal |
| **Print — business card** | 45–55mm | Primary horizontal |
| **Print — large format (poster/banner)** | Scale to context, maintain 2× clear space | Primary horizontal or stacked |
| **Favicon** | 32×32px (with 16×16 fallback) | Simplified icon |

## Pairing with Brand Platform Text

"Torne-se Chainless." is the brand platform and frequently appears alongside the logo. Rules for pairing:

### Logo + "Torne-se Chainless."

- The brand platform text is set in **Fraunces** (Optical, italic) at a size that creates visual hierarchy below the logo — never competing with the wordmark.
- **Placement:** Below the primary or stacked lockup, optionally separated by a fine rule (`1px, #E5E3DF`).
- **Size relationship:** The platform text should be approximately 40–60% of the wordmark size.
- **Never** set "Torne-se Chainless." in the same typeface as the wordmark. The serif/sans contrast is intentional — it introduces the 30% rebel energy at the logo level.

### "Torne-se Chainless." Standalone

When the platform text appears without the logo (e.g., campaign hero, social posts):
- Set in **Fraunces**, italic, medium weight
- Size: Display-level (48–72px) for hero, H2–H3 level for supporting contexts
- Color: `#1C1B19` on light, `#FAFAF8` on dark, or `#FFC73D` on dark for high-impact moments
- Always followed by a period: "Torne-se Chainless**.**" — the period is part of the expression. It conveys conviction, not trailing off.

## Co-Branding & Partnership Usage

### With Partner Logos

- Maintain equal visual weight (scale to match perceived size, not pixel dimensions)
- Separate with a vertical divider (`1px, #D1CFC9`) with 2× clear space on each side
- Both logos on the same baseline
- Never modify the Chainless logo to match a partner's style
- **Chainless logo always appears first** (left in LTR contexts) unless co-branding agreement specifies otherwise

### Integration Partners (DeFi protocols, payment rails)

- Partner logos appear in a "Powered by" or "Integrations" section, never alongside the Chainless logo in primary lockup position
- Partner logos rendered in monochrome (grayscale) to maintain Chainless visual authority
- Exceptions: co-marketing materials approved by brand team

## Logo Don'ts

1. **Don't distort.** Never stretch, squeeze, rotate, or skew the logo.
2. **Don't recolor.** Use only approved color variations (dark mono, light mono, on brand yellow). Never apply gradients, patterns, drop shadows, or effects.
3. **Don't add elements.** No taglines, icons, or decorative elements within the clear space.
4. **Don't place on busy backgrounds.** If a photographic background is necessary, ensure sufficient contrast using a subtle overlay or placing the logo in a clear area.
5. **Don't recreate.** Always use official logo files — never retype the wordmark or redraw the icon.
6. **Don't animate the logo.** The logo mark itself is static. Animations may surround or introduce the logo but never deform, morph, or particle-ize it.
7. **Don't use the icon as a bullet point or decorative element.** The icon is the brand mark, not a graphic device.
8. **Don't outline.** Never use the logo as a stroke-only/outline treatment.

### Background Rules

| Background | Approved | Notes |
|------------|:--------:|-------|
| `#FAFAF8` warm off-white | Yes | Default — dark mono logo |
| `#F4F3F0` secondary bg | Yes | Dark mono logo |
| `#FFC73D` brand yellow | Yes | Dark mono logo — premium brand moment |
| `#1C1B19` warm dark | Yes | Light mono logo |
| `#000000` pure black | Yes | Light mono logo |
| Photography (light) | Conditional | Only if clear space has sufficient contrast |
| Photography (dark) | Conditional | Light mono logo, ensure contrast |
| Gradients | No | Never place logo on gradient backgrounds |
| Patterns/textures | No | Never — the logo needs clean space |

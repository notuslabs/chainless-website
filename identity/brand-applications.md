# Brand Applications

> Phase: identity | Brand: chainless | Generated: 2026-03-22

---

## Design Philosophy

Every touchpoint is a brand moment. The identity system only works when it's applied consistently across contexts — digital and physical. These applications demonstrate how the logo system, color composition, typography pairing, and imagery direction converge in real-world use.

The governing principle: **the 60-30-10 color composition + Söhne/Fraunces typography pairing + warm imagery = the Chainless feel.** If any application doesn't immediately read as "warm, confident, sovereign," it needs revision.

## Digital Touchpoints

### 1. Website — Homepage Hero

The most important 5 seconds of brand experience. The hero is where the Magician speaks loudest.

| Element | Specification |
|---------|---------------|
| **Background** | Dark surface `#1C1B19` — INVERTED section. Full viewport height (100vh) or near-full. |
| **Headline** | "Torne-se Chainless." in Fraunces Bold Italic, Display size (72px desktop / 44px mobile), `#FAFAF8`. Centered or left-aligned depending on layout. |
| **Subheadline** | "Seu patrimônio cresce. Suas chaves continuam suas." in Söhne Light, H4 size (28px), `#B0ADA6`. Below headline with 24px spacing. |
| **CTA** | "Começar agora" — Söhne Semibold, 18px. Background `#FFC73D`, text `#1C1B19`. Pill shape (full radius), 56px height, 32px horizontal padding. |
| **Secondary action** | "Entenda autocustódia →" — Söhne Medium, 16px, `#FFC73D` text with underline animation on hover. |
| **Visual** | Subtle background: warm gradient from `#1C1B19` to `#2A2926`, or a desaturated lifestyle photograph with dark overlay (60% opacity). Abstract data visualization element (yield curve) in `#FFC73D` at 20% opacity as atmospheric background. |
| **Motion** | Headline fades in (600ms, ease-out). CTA slides up (400ms, ease-out, 100ms delay). Background visual has subtle parallax on scroll. |

| Type | Direction |
|------|-----------|

## Page Layout: 60-30-10 in Practice

A complete homepage section breakdown showing color composition in action:

```
Section                    BG Color        Text Color     Accent          Composition
─────────────────────────────────────────────────────────────────────────────────────
Nav                        #FAFAF8         #1C1B19        #FFC73D (CTA)   60% light
Hero                       #1C1B19         #FAFAF8        #FFC73D         INVERTED
Trust bar (logos)          #F4F3F0         #9C9890        —               60% light
Value props                #FAFAF8         #1C1B19        #FFC73D (stats) 60% light
Feature deep-dive          #1C1B19         #FAFAF8        #FFC73D         INVERTED
Social proof               #FAFAF8         #1C1B19        —               60% light
How it works               #F4F3F0         #1C1B19        #FFC73D (steps) 60% light
CTA section                #1C1B19         #FAFAF8        #FFC73D (btn)   INVERTED
Footer                     #1C1B19         #B0ADA6        #FFC73D (links) INVERTED
```

**Result:** 5 light sections, 4 dark sections. The alternation creates rhythm. Yellow appears in every section but never dominates. The page reads as warm, premium, and confident.

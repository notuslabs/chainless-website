# Brand Book Outline

> Phase: identity | Brand: chainless | Generated: 2026-03-22

---

## Overview

A 20-page brand book that serves as the definitive guide for anyone creating for Chainless — internal team, agency partners, and contractors. Every page reinforces one principle: **warm, confident, sovereign.**

The book is designed with the same visual system it documents: Söhne body, Fraunces display accents, warm neutrals with yellow highlights, generous whitespace, premium paper stock (if printed).

---

## Page-by-Page Outline

### Pages 1–2: Cover + Brand Platform

**Cover (Page 1)**
- Full-bleed `#FFC73D` background
- Chainless logo centered, `#1C1B19`
- "Brand Guidelines" in Söhne Light, small, bottom-center
- Design notes: The yellow cover IS the brand statement — confident, warm, unmistakable. No photograph, no pattern. Just gold.

**Brand Platform (Page 2)**
- "Torne-se Chainless." in Fraunces Bold Italic, Display size, `#1C1B19` on `#FAFAF8`
- Purpose statement below: "We believe your wealth should answer to you — and only you."
- Design notes: One line. Maximum whitespace. The brand platform gets a page to itself because it IS the brand.

### Pages 3–4: Brand Story + Strategy Recap

**Brand Story (Page 3)**
- Narrative: the origin of Chainless, the problem it solves, the future it envisions
- Written in the brand voice (confident, warm, clear) — not corporate third-person
- Pull quote in Fraunces italic: "Self-custody isn't a feature. It's the architecture."
- Design notes: Editorial layout. Generous margins. One column of text with a sidebar pull quote.

**Strategy at a Glance (Page 4)**
- Archetype blend: Magician (70%) + Ruler (30%) — visual diagram
- Positioning: Premium × Self-Custody quadrant map
- Brand platform: "Torne-se Chainless."
- Values: five values listed with one-line descriptions
- Design notes: Infographic-style. Clean grid. Icons for each value. The strategy distilled to a scannable reference page.

### Pages 5–6: Voice & Tone

**Voice Overview (Page 5)**
- The 60/40 split visualized: premium warmth vs. defiant counter-positioning
- Four voice attributes: Confident, Defiant, Warm, Clear — with one-line definitions
- Tone spectrum visualization
- Design notes: The voice section is designed as a reference card — something a copywriter can pin up.

**Do's and Don'ts — Copy (Page 6)**
- Side-by-side headline examples (do/don't) in both PT-BR and EN
- Forbidden words list with alternatives
- Microcopy examples for product UI
- Design notes: Two-column layout. Green checkmark (success-500) for do's, red X (error-500) for don'ts. Real examples, not abstractions.

### Pages 7–9: Logo System

**Logo Lockups (Page 7)**
- All approved lockups: primary horizontal, stacked, icon only, monochrome variations
- Clear space diagram with cap-height measurement grid
- Minimum sizes table
- Design notes: White background. Logo rendered large with measurement annotations. Technical but beautiful.

**Logo in Context (Page 8)**
- Logo on approved backgrounds: light, dark, yellow, photography
- Responsive sizing guidelines table
- Co-branding layout example
- Design notes: Grid of context photos showing the logo in real-world applications.

**Logo Don'ts (Page 9)**
- Grid of incorrect applications: distorted, wrong colors, busy background, recreated, outlined, animated incorrectly
- Each violation labeled with a brief explanation
- Design notes: Subtle red overlay or strike-through on each violation. Immediately scannable.

### Pages 10–12: Color System

**Primary Palette (Page 10)**
- Brand Yellow scale (50–950) as color swatches with hex and OKLCH values
- Neutral scale (50–950) as color swatches
- "Source" indicator on `#FFC73D` (yellow-500) and `#1C1B19` (neutral-500)
- Design notes: Full-width swatches. Each swatch large enough to see the color clearly. Hex code below each in IBM Plex Mono.

**Extended Palette + Semantics (Page 11)**
- Warm neutral scale (the 60% backbone)
- Semantic colors: success, warning, error, info — with swatches and usage labels
- Dark mode mapping diagram: light swatch → arrow → dark mode equivalent
- Design notes: Two-column layout. Left: warm neutrals. Right: semantics + dark mode.

**Color Composition (Page 12)**
- 60-30-10 rule diagram with proportional color blocks
- Inverted sections diagram (page rhythm visualization)
- Contrast ratio reference table for common combinations
- Design notes: This is the "how to use color" page. Practical, not theoretical. Shows the page rhythm pattern visually.

### Pages 13–14: Typography

**Type System (Page 13)**
- Söhne specimen: alphabet, weights (Extralight through Extrabold), a short paragraph
- Fraunces specimen: alphabet, weights, "Torne-se Chainless." set in its intended style
- IBM Plex Mono specimen: alphabet, code example, data display
- Design notes: Large type specimens. Let the letterforms speak. Minimal annotation — the type IS the content.

**Type Scale + Pairing Rules (Page 14)**
- Full type scale table: level, px, weight, line-height, font, use case
- Pairing decision tree: "Is this a brand moment? → Fraunces. Is this a product moment? → Söhne. Is this data? → IBM Plex Mono."
- Responsive behavior summary
- Design notes: Reference-card format. The type scale table is the most-referenced page in any brand book — make it scannable and precise.

### Pages 15–16: Imagery

**Photography Direction (Page 15)**
- Mood board of approved photography styles: warm light, hands on devices, considered environments, over-shoulder views
- Color grading specifications: white balance, saturation, contrast, highlights
- Design notes: Grid of 6–9 reference images. Each with a one-line caption describing what works about it.

**Photography Don'ts + Iconography (Page 16)**
- Grid of rejected styles: stock clichés, crypto clichés, luxury signaling, cold corporate
- Icon style specifications: stroke weight, grid, corner radius
- Phosphor Icons reference set for common use cases (20–30 key icons)
- Design notes: Two-thirds photography don'ts, one-third iconography spec.

### Pages 17–18: Applications

**Digital Applications (Page 17)**
- Website hero screenshot/mockup showing full identity in action
- Dashboard view (cropped) showing data visualization, typography, color in product
- Social media card example (dark card, Fraunces headline, yellow accent)
- Design notes: These are aspirational mockups — they show what Chainless looks like when everything comes together. Full-bleed, beautiful.

**Physical Applications (Page 18)**
- Business card (front and back)
- Presentation slide (title + content)
- Merchandise (t-shirt, sticker, notebook)
- Design notes: Photography of physical items (or high-fidelity mockups). Show the brand in three dimensions.

### Pages 19–20: Brand in Action + Closing

**Brand in Action (Page 19)**
- Full-page spread showing the homepage layout with annotations: "60-30-10 composition," "Fraunces hero → Söhne sections," "inverted dark sections," "yellow as 10% accent"
- A visual "proof" that the system works together
- Design notes: The most complex page — annotation-rich but still beautiful. Shows the system, not just the pieces.

**Closing (Page 20)**
- "Torne-se Chainless." in Fraunces Bold Italic, centered, `#1C1B19` on `#FAFAF8`
- Contact for brand questions: brand@chainless.finance (or appropriate contact)
- Version number and date: "v1.0 — March 2026"
- Design notes: Mirror the brand platform page (page 2). The book opens and closes with the same statement — bookending the identity with its organizing principle.

## Strategic Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Category | "Sovereign digital wealth" (new category creation) | Premium self-custody quadrant is empty globally. Don't compete in existing categories — define a new one. |
| Archetype | Magician (70%) + Ruler (30%) | Magician makes self-custody aspirational ("Torne-se Chainless"). Ruler makes it credible (premium, architectural). |
| Voice | Bold, Defiant, Warm — 60/40 premium/rebel | Voice leads with more edge than visual system (60/40 vs 70/30). Intentional tension between verbal defiance and visual warmth. |
| Brand platform | "Torne-se Chainless." elevated from buried CTA to organizing principle | Already has equity. Identity-level transformation language. Scales across hero, CTA, community, content, journey. |
| Tagline hierarchy | "Torne-se Chainless." (primary) → "Seu patrimônio. Suas chaves." (supporting) → "Patrimônio digital soberano." (category) | Three layers: aspirational → descriptive → categorical. |
| Message hierarchy | Sovereignty → Wealth growth → Simplicity → Features | Features are proof points, never headlines. Self-custody IS the brand, not a feature. |
| Counter-positioning | Architectural, not rhetorical — quiet confidence, never aggression | Self-custody IS the counter-position. Target systems, never people. "We built something they can't" not "they're the enemy." |

## Key Takeaways for Downstream Phases

### For Identity Phase
- Archetype blend is Magician 70% / Ruler 30% — visual system should feel transformative AND premium, not one or the other
- Warm premium (Stripe/Notion school) is the dominant aesthetic — but with 20% self-sovereignty branding and 10% counter-positioning spice
- "Torne-se Chainless" needs typographic treatment as the brand platform — consider accent/display type for these moments
- Light mode default signals "wealth platform" not "trading floor" — counter-positions against crypto dark-mode-everything
- Preserve logo, color palette, "Chainless" name. Everything else can evolve.

### For System Phase
- Voice attributes (Confident, Defiant, Warm, Clear) must be reflected in component microcopy defaults
- Tone shifts by context — components need to carry the right tone for their location (onboarding = warm, dashboard = authoritative)
- Premium whitespace is a brand signal — spacing tokens should be generous
- Words-we-use/don't-use table should inform content linting or review processes

### For Project Phase (Website)
- Message hierarchy (sovereignty → growth → simplicity → features) drives page architecture
- Homepage hero leads with sovereignty, not features
- "Torne-se Chainless" is the primary CTA — conversion through identity, not "Sign Up"
- Ricardo path: Ruler-first (premium, trust) → Magician (transformation). Marina path: Magician-first (vision) → Ruler (architecture quality).
- Pix is footnote-level. Self-custody is headline-level.
- Counter-positioning moments should be strategic and sparse — not every section, but the sections that matter

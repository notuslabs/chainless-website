# Verbal Identity

> Brand: chainless | Completed: 2026-03-22

## Summary

Verbal identity defines how Chainless speaks — the voice attributes, tone spectrum, messaging hierarchy, and language rules that govern every word the brand produces. The voice operates on a deliberate 60/40 tension: premium warmth / defiant counter-positioning.

## Key Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Voice split | 60% premium warmth / 40% defiant counter-positioning | Voice leads with more edge than visual system (60/40 vs 70/30). Intentional tension. |
| Voice attributes | Confident, Defiant, Warm, Clear | Four attributes that define every piece of Chainless communication. |
| Core message | "Your wealth grows. Your keys stay yours." | Irreducible truth — two promises, one sentence, architecturally guaranteed. |
| Message hierarchy | Sovereignty -> Wealth growth -> Simplicity -> Features | Features are proof points, never headlines. Self-custody IS the brand. |
| Tagline hierarchy | "Torne-se Chainless." (primary) -> "Seu patrimonio. Suas chaves." (supporting) -> "Patrimonio digital soberano." (category) | Three layers: aspirational -> descriptive -> categorical. |
| Language | PT-BR primary, modern Brazilian Portuguese | Not formal/literary. Natural, warm, direct. |
| Forbidden words | "Banco," "WAGMI," "Confie em nos," "Facil," "Melhor," "Revolucionario" | Regulatory risk, crypto-bro culture, contradiction of brand thesis, commodity language, unsubstantiated claims, hyperbole. |

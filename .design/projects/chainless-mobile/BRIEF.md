# Project Brief: chainless-mobile

**Brand:** chainless
**Date:** 2026-04-08
**Scope:** Mobile responsiveness audit and fixes for the Chainless landing site

## Goal

Audit all landing page sections at mobile viewports (320px–640px) and fix responsiveness issues: layout overflows, inconsistent padding, touch target sizes, typography scaling, video performance, scroll UX, and spacing rhythm. Ship mobile as a first-class experience matching the desktop premium feel.

## Platform

Web — responsive mobile (iOS Safari, Android Chrome)

## Tech Stack

Next.js 16 + Tailwind v4 + Framer Motion 12

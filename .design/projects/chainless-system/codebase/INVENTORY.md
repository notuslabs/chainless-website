## Notable Observations

1. **Font licensing gap:** Identity spec calls for Sohne (Klim, paid) — implementation uses Satoshi (FontShare, free). General Sans as bridge fallback.
2. **Dark-only implementation:** Despite identity spec suggesting light-mode default ("signals wealth platform"), the landing is entirely dark-theme. Light surface tokens exist in CSS but are unused.
3. **Dead code:** `spend-credit-legacy.tsx` is not imported — was replaced by `spend-credit.tsx` (video version).
4. **Heavy animation:** Every section uses Framer Motion. Two components do manual canvas/rAF rendering (editorial-break, social-proof).
5. **Accessibility signals present:** `aria-label`, `aria-labelledby`, `aria-live`, `role="status"`, `sr-only`, `useReducedMotion()`, semantic HTML, focus-visible rings globally styled.
6. **No data layer:** All content hardcoded in components. Email capture form simulates submission with setTimeout.
7. **Single-page architecture:** Only one route (`/`). 12 sections composed sequentially in page.tsx.
